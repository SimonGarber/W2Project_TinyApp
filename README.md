# W2Project_TinyApp


# *Tiny App Project*

Tiny app is a fully stack web application built with node and express that allows users to shorted long URLS (a la Bit.ly). 



## *Dependancies* 

- node.js
- express
- Ejs
- bcrypt 
- body parser
- cookie session

- Install all dependancies (using the npm install command)
